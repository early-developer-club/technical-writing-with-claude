# Posts Package

MDX 기반 블로그 포스트 저장소입니다.

## 구조

- `drafts/`: 작성 중인 초안
- `published/`: 게시된 포스트

## 포스트 작성 가이드

### Frontmatter 예시

```yaml
---
title: "포스트 제목"
description: "포스트 설명"
date: "2025-10-16"
updated: "2025-10-16"
tags: ["Next.js", "React"]
category: "Frontend"
author: "작성자"
status: "published"
featured: false
---
```

### 파일명 규칙

- 영문 소문자와 하이픈 사용
- 예: `nextjs-15-new-features.mdx`
