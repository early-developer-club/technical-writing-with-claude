#### `transform()` 🟡

```typescript
function transform<T, U>(
  input: T,
  transformer: (value: T) => U
): U
```

**용도**: 데이터를 변환합니다.

**예제**:
```typescript
const num = transform('42', str => parseInt(str));

```
